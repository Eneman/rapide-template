<h1 align="center"> Rapide Template </h1>

This template provides a minimal setup to get React working in Vite.

It includes the following tools:
- Vite 6.2
- React 19.0
- Typescript 5.7
- Tailwind 4.0
- Vitest 3.0
- Biome 1.9 (replace ESLint and Prettier)

## Installation

### Using ```degit```
```
npx degit Eneman/rapide-template#master <PROJECT_NAME>
cd <PROJECT_NAME>

npm install
npm run dev
```
### Using ```git clone```
```
git clone https://github.com/Eneman/rapide-template <PROJECT_NAME>
cd <PROJECT_NAME>

npm install
npm run dev
```
